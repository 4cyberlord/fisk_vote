## Verification Checklist

Before finalizing any name, you MUST check:

### 1. Domain Availability
- Check: [Namecheap](https://www.namecheap.com/)
- Check: [GoDaddy](https://www.godaddy.com/)
- Check: [Google Domains](https://domains.google/)

### 2. Trademark Search
- USPTO Database: https://www.uspto.gov/trademarks/search
- Search exact name and variations

### 3. GitHub/Social Media
- GitHub: Search for repositories
- Twitter: Check @handle availability
- LinkedIn: Check company page
- Facebook: Check page availability

### 4. Business Name Search
- State business registry
- Local business databases

### 5. App Store Search
- Apple App Store
- Google Play Store
